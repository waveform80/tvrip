===========
mkvpropedit
===========

::

    set mkvpropedit /path/to/mkvpropedit

Sets the path of the mkvpropedit binary, used by tvrip to fill out the MKV
meta-data fields after ripping is complete. Example::

    (tvrip) set mkvpropedit /usr/bin/mkvpropedit
