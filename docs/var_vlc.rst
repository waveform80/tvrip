===
vlc
===

::

    set vlc /path/to/vlc

Sets the path of the VLC binary, used by tvrip to play back titles either on
demand or during auto-mapping of chapters.

Example::

    (tvrip) set vlc /usr/bin/vlc

See also :doc:`cmd_play`, :doc:`cmd_automap`
