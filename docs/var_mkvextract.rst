===========
mkvextract
===========

::

    set mkvextract /path/to/mkvextract

Sets the path of the mkvextract binary, used by tvrip to fill out the MKV
meta-data fields after ripping is complete. Example::

    (tvrip) set mkvextract /usr/bin/mkvextract
