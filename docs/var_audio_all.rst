=========
audio_all
=========

::

    set audio_all <bool>

This setting determines whether all audio tracks matching the selected
languages are to be ripped (when set to ``on``), or just the "best", as
determined by audio mix (when set to ``off``).

See also :doc:`var_audio_langs`
